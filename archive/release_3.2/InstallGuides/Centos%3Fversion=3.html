Chaste installation on Centos

These are the steps that I followed to install all the Chaste dependencies, run the test pack and build the executable.

Details of the machine are:

Linux version 2.6.18-164.10.1.el5. Red Hat 4.1.2-46

Preliminary

Install gcc, g++, libz and forrtan compiler (the CentOS operating system apparently comes with none of these). 

sudo yum install gcc 
sudo yum install gcc-c++
sudo yum install zlib-devel
sudo yum install gcc-gfortran

Scons

Download tarball from  http://prdownloads.sourceforge.net/scons/scons-1.2.0.tar.gz

Then: 

tar -xzvf scons-1.2.0.tar.gz
cd scons-1.2.0
python setup.py install --prefix=$HOME

Petsc (version 3)

Download the tarball from  http://www.mcs.anl.gov/petsc/petsc-as/download/index.html. Petsc 3 takes care of downloading, compiling and installing blas, lapack, openmpi, hdf5, hypre 

tar -xzvf petsc-3.0.0-p11.tar.gz
cd petsc-3.0.0-p11
export PETSC_DIR=`pwd`
export PETSC_ARCH=linux-gnu
./config/configure.py --with-gnu-compilers==true --download-f-blas-lapack=1 --with-x=false --with-clanguage=cxx --download-hypre=yes --download-openmpi=yes --CXXFLAGS=-fPIC --CFLAGS=-fPIC --download-hdf5=yes
make all test

export PETSC_ARCH=linux-gnu-opt
./config/configure.py --with-gnu-compilers==true --download-f-blas-lapack=1 --with-x=false --with-clanguage=cxx --download-hypre=yes --download-openmpi=yes --CXXFLAGS=-fPIC --CFLAGS=-fPIC --with-debugging=0 --download-hdf5=ifneeded
make all test

Add to the PATH the mpicc inside the petsc/linuxgnu

XML Tools

The CentOs yum repository seems not to have xerces-c libraries. Either you change the yum config file and add more repositories or download xerces-c and xerces-c-devel from  http://dag.wieers.com/rpm/packages/xerces-c/ and then 

sudo rpm -i xerces-c-2.7.0-1.2.el4.rf.i386.rpm
sudo rpm -i xerces-c-devel-2.7.0-1.2.el4.rf.i386.rpm

For xsd (assuming our installation directory is ~/Chaste 

cd ~/Chaste
wget http://codesynthesis.com/download/xsd/2.3/linux-gnu/i686/xsd-2.3.1-i686-linux-gnu.tar.bz2
tar -xjf xsd-2.3.1-i686-linux-gnu.tar.bz2
ln -s ~/Chaste/xsd-2.3.1-i686-linux-gnu/bin/xsd ~/Chaste/bin/xsd

Boost

Download the boost-jam rpm from  http://sourceforge.net/projects/boost/files/boost-jam/ 

rpm -i boost-jam-3.1.12-1.i386.rpm

Download boost (1.37) from  http://sourceforge.net/project/showfiles.php?group_id=7586&package_id=8041 

tar -zxvf boost_1_37_0.tar.gz
cd boost_1_37_0
bjam "-sTOOLS=gcc" --prefix=$HOME/boost install

METIS

cd $HOME
wget http://glaros.dtc.umn.edu/gkhome/fetch/sw/metis/metis-4.0.tar.gz
tar -zxf metis-4.0.tar.gz

Edit the file Makefile.in, changing the line CC = cc to CC = gcc.

Then run 

make
